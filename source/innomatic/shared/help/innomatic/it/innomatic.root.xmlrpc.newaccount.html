Nuovo account di un profilo di servizi web ========================================== Da questa pagina è possibile creare un nuovo account di un profilo di servizi web di un altro host. Se non vengono indicati il nome utente e la password, Innomatic crea un utente anonimo. Si consiglia di porre particolare attenzione nel scegliere quale profilo associare all'utente anonimo, perchè chiunque può accedere ai servizi web associati al profilo di un utente anonimo.
